##  Disciplined Trader

     Trade the plan. Master the discipline.

A high-density journaling and signal verification platform engineered to eliminate emotional bias through automated checklist enforcement and multi-timeframe analysis.

##  Technical Stack

    Frontend: Next.js 16 / TypeScript / Tailwind CSS

    Backend: TypeScript / Node.js / High-Performance API

    Database: PostgreSQL (Neon)  / Prisma

    Auth: Better Auth
   

 Core Systems

    Hard-Stop Checklist: Circuit-breaker logic that prevents trade logging unless strategy criteria are met.

    Signal Matrix: Real-time M15–Daily bias aggregation.

    Behavioral Alpha: Automated adherence scoring (0-100%) to track strategy drift.

    No-Noise UI: Designed for peak cognitive focus during live market hours.
